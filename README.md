# p5.js 選擇題測驗系統

## 功能特色

- 📊 **CSV題庫支援**: 使用CSV檔案作為題庫，方便編輯和管理
- 🎨 **p5.js圖形介面**: 美觀的互動式測驗介面
- ✅ **即時回饋**: 答題後立即顯示正確答案和詳細解析
- 📈 **分數統計**: 即時顯示答題進度和分數
- 🔄 **重複測驗**: 完成後可重新開始測驗

## 檔案結構

```
├── index.html          # 主要HTML檔案
├── sketch.js           # p5.js主程式
├── questions.csv       # 題庫檔案（CSV格式）
├── style.css           # 樣式檔案（未使用）
└── libraries/
    ├── p5.min.js       # p5.js核心函式庫
    └── p5.sound.min.js # p5.js音效函式庫
```

## CSV題庫格式

題庫檔案 `questions.csv` 的格式如下：

```csv
題目,選項A,選項B,選項C,選項D,正確答案,解析
什麼是JavaScript中的變數宣告關鍵字？,var,const,let,以上皆是,D,JavaScript中var、const、let都是變數宣告關鍵字
```

### 欄位說明

- **題目**: 問題內容
- **選項A-D**: 四個選擇項目
- **正確答案**: 正確選項（A、B、C或D）
- **解析**: 答案說明

## 使用方式

### 1. 準備題庫
編輯 `questions.csv` 檔案，按照格式添加題目。可以使用Excel或其他試算表軟體編輯，然後存成CSV格式。

### 2. 啟動測驗
在瀏覽器中開啟 `index.html` 檔案即可開始測驗。

### 3. 答題流程
1. 閱讀題目
2. 點擊選擇答案
3. 點擊任意位置查看解析
4. 5秒後自動進入下一題
5. 完成所有題目後查看總分

## 自訂設定

### 修改畫布大小
在 `sketch.js` 的 `setup()` 函數中修改：
```javascript
let canvas = createCanvas(800, 600); // 寬度, 高度
```

### 修改自動進入下一題的時間
在 `drawExplanation()` 函數中修改：
```javascript
let remainingTime = 5 - floor((millis() - explanationTimer) / 1000); // 5秒
```

### 修改顏色主題
在 `draw()` 函數中修改背景色：
```javascript
background(240, 248, 255); // R, G, B值
```

## 技術特點

- 使用p5.js的 `loadTable()` 函數載入CSV檔案
- 響應式按鈕設計，支援滑鼠懸停效果
- 自動換行功能，支援長題目顯示
- 狀態機設計，清晰的游戲流程控制

- Safari

## 故障排除

### 題目無法載入
- 檢查 `questions.csv` 檔案是否存在
- 確保CSV格式正確
- 檢查是否在HTTP伺服器環境中執行

### 畫面顯示異常
- 檢查瀏覽器開發者工具的錯誤訊息
- 確保所有檔案路徑正確
- 檢查p5.js函式庫是否正確載入
